# on-igiri-releases

Public release mirror for [Onigiri](https://github.com/joan-chiangwq/on-igiri)
(private source repo). This repo contains **no source code** — only:

- **Releases** — signed, notarized `.dmg` builds, attached as GitHub Release
  assets.
- **`appcast.xml`** — the Sparkle auto-update feed. Installed copies of
  Onigiri read this file (via `raw.githubusercontent.com`) to check for new
  versions.

This repo exists because GitHub only serves release assets and raw file
content anonymously from **public** repos — `on-igiri` itself stays private,
so its release artifacts are mirrored here instead.

Published and updated automatically by `on-igiri`'s `scripts/release.sh`.
Don't edit `appcast.xml` by hand — it's regenerated by Sparkle's
`generate_appcast` tool each release.
